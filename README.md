# 🌌 VYOM BOLD

**Vyom Bold** (Sanskrit: *Vyom* meaning Sky/Infinite) is a minimalist, high-performance music player built with Python. It bridges the gap between celestial aesthetics and local audio playback, offering a seamless experience for music lovers who value both style and stability.

![Vyom Bold Banner](https://your-placeholder-link.com/banner.png)

## ✨ Key Features

* **Infinite Interface:** A deep-space inspired dark theme with Neon Cyan accents.
* **Smart Playback:** Supports MP3 and WAV files with automatic metadata and album art extraction.
* **Playlist Management:** * **Drag & Add:** Easily upload entire folders or single tracks.
    * **Right-Click to Remove:** Instant queue cleanup from the playlist tab.
* **Playback Modes:**
    * 🔀 **Shuffle:** Randomize your listening experience with a single click.
    * 🔁 **Repeat Modes:** Cycle between *Repeat Off*, *Repeat All*, and *Repeat One*.
* **Volume Precision:** Smooth slider control for perfect audio leveling.
* **Stability First:** Built with a decoupled audio engine to prevent UI freezing.
